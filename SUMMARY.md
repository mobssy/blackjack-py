# 🎰 JackPy 프로젝트 요약

## 📋 프로젝트 개요

**JackPy**는 텔레그램 기반의 프로덕션급 블랙잭 카지노 봇으로, 실제 상업 서비스를 목표로 개발되었습니다. 무료/VIP/비즈니스 플랜 구조를 가진 수익형 SaaS 플랫폼입니다.

---

## ✅ 구현 완료 항목

### 1️⃣ 데이터베이스 모델 (models/)

#### ✅ User 모델 (models/user.py)
- 사용자 정보 관리
- VIP 상태 및 만료일 관리
- 잔액 관리 (Decimal 타입)
- 게임 통계 (JSON)
- 데일리 보상 시스템

#### ✅ Group 모델 (models/group.py)
- 텔레그램 그룹 관리
- 플랜 타입 (FREE/VIP/BUSINESS)
- 플랜 만료 관리
- 커스텀 설정 (JSON)

#### ✅ Round 모델 (models/round.py)
- 게임 라운드 기록
- 플레이어/딜러 핸드 저장
- 게임 결과 및 정산 금액
- 타임스탬프

#### ✅ Approval 모델 (models/approval.py)
- VIP/비즈니스 승인 요청
- 입금 정보 관리
- 승인/거절 상태 관리

#### ✅ AdSchedule 모델 (models/ad_schedule.py)
- 그룹별 광고 스케줄
- 광고 발송 간격 관리

---

### 2️⃣ 유틸리티 함수 (bot/utils/)

#### ✅ deck.py - 카드 덱 관리
- `Card` 클래스: 카드 표현 및 값 계산
- `Deck` 클래스: 멀티덱 지원, 셔플, 카드 뽑기
- `calculate_hand_value()`: Soft/Hard Ace 처리
- `is_blackjack()`: 블랙잭 판정
- `is_bust()`: 버스트 판정
- `format_hand()`: 카드 표시 (이모지 지원)

#### ✅ payouts.py - 정산 로직
- `PayoutCalculator`: 결과별 정산 금액 계산
  - 블랙잭: 3:2 배당
  - 일반 승리: 1:1 배당
  - 무승부: 베팅액 반환
- `determine_outcome()`: 게임 결과 판정
- 결과 메시지 및 이모지 생성

#### ✅ ads.py - 광고 시스템
- `AdManager`: 광고 표시 관리
- 확률 기반 광고 노출 (기본 15%)
- 무료 플랜 전용
- 랜덤 광고 메시지 선택

#### ✅ scheduler.py - 스케줄러
- `JackPyScheduler`: APScheduler 기반
- VIP 만료 체크 및 알림 (3일/1일/당일 전)
- 그룹 플랜 만료 체크
- 자동 상태 업데이트

### 3️⃣ 게임 핸들러 (bot/handlers/)

#### ✅ blackjack.py - 블랙잭 게임
- `/deal [금액]`: 게임 시작
  - 잔액 확인
  - 초기 카드 딜
  - 블랙잭 자동 체크
- `/hit`: 카드 추가
  - 버스트 자동 체크
- `/stand`: 멈춤
  - 딜러 자동 플레이 (17 이상)
  - 결과 판정 및 정산
- `/double`: 더블 다운 (첫 두 장, 베팅 2배 + 카드 1장 후 자동 스탠드)
- `/surrender`: 서렌더 (첫 두 장, 베팅액 절반 회수)
- `/split`: 스플릿 (같은 랭크 2장을 두 핸드로 분리, 추가 베팅)
- `/wallet`: 잔액 및 통계 확인
- `/daily`: 일일 보상
  - 무료: $200
  - VIP: $500

#### ✅ vip.py - VIP 및 비즈니스 플랜
- `/vip`: VIP 플랜 안내
  - 가격표 (30/90/365일)
  - 혜택 안내
  - 입금 계좌 정보
- `/business`: 비즈니스 플랜 안내
- `/confirm [입금자명] [금액] [기간]`: VIP 입금 확인
  - 승인 요청 생성
  - 관리자 자동 알림
- `/confirm_business [입금자명] [금액]`: 비즈니스 입금 확인

#### ✅ admin.py - 관리자 기능
- `/admin pending`: 승인 대기 목록
- `/admin stats`: 전체 통계
- `/approve [user_id] [days]`: VIP 승인
- `/approve_business [user_id] [chat_id]`: 비즈니스 승인
- `/reject [user_id] [사유]`: 승인 거절
- `/revoke [user_id]`: VIP 해제

#### ✅ profile.py - 프로필 및 랭킹
- `/my`: 프로필 카드
  - 잔액, VIP 상태
  - 게임 통계 (승률, 총 게임 수)
  - 재무 정보
- `/rank`: 잔액 순위 (Top 10)
- `/stats`: 상세 통계
  - 최대 베팅/승리
  - 총 수익

#### ✅ start.py - 시작 및 도움말
- `/start`: 봇 시작
  - 신규 사용자 자동 등록 ($1,000 지급)
  - 플랜 안내 카드
  - 인라인 버튼
- `/help`: 전체 명령어 안내
- 콜백 쿼리 핸들러
